# 🌊 Hostel Roots — Painel do Projeto

Painel de gestão do projeto **Hostel Roots**, um hostel temático (Reggae / Rock / Surf) em Aracaju-SE.

## 📋 Conteúdo

O painel possui 5 abas:

| Aba | Descrição |
|-----|-----------|
| **Visão Geral** | KPIs principais, localização e diferenciais |
| **Cronograma** | 4 fases com tarefas interativas e barra de progresso |
| **Financeiro** | Investimento inicial, custos mensais e cenários |
| **Conceito** | Alas temáticas e experiências extras |
| **📈 Receita** | Estratégias de alavancagem e projeção de 12 meses |

## 💰 Resumo Financeiro

| Item | Valor |
|------|-------|
| Investimento inicial | ~R$ 79.000 |
| Custo fixo mensal | ~R$ 16.650 |
| Receita potencial (hospedagem) | ~R$ 25.000/mês |
| Receita total (+ ancillary) | ~R$ 52.000/mês |
| Lucro líquido estimado | ~R$ 28.000/mês |
| Break-even | ~5 meses |

## 🗺️ Localização

- **1ª opção:** Coroa do Meio (custo-benefício + proximidade orla)
- **Alternativa:** Atalaia (maior visibilidade turística)
- **Vibe surf:** Aruana (mais exclusivo e tranquilo)

## 🚀 Como usar

Abra o arquivo `index.html` em qualquer navegador. Nenhuma dependência externa necessária além de conexão para fontes Google Fonts.

## 📈 Alavancas de Receita

1. **Precificação dinâmica** — diária sobe até R$ 250 em alta temporada
2. **Bar Roots** — receita estimada de R$ 18K/mês
3. **Pacotes premium** — Surf Week, Roots Weekend, passeios
4. **Eventos externos** — abre espaço para público de fora
5. **Expansão gradual** — de 20 para 35 camas sem grande reforma
